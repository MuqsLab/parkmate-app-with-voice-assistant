# ParkMate Assessment 3 Prototype

This React/Vite proof-of-concept demonstrates an advanced **Voice Parking Assistant** feature for the ParkMate mobile app concept.

## Feature summary

The prototype includes:

- Text-to-speech parking guidance using the browser SpeechSynthesis API
- Live Melbourne time in the phone status bar
- Scrollable mobile phone-style interface
- Interactive Map, Favourites, and Safety tabs
- Availability cards with visual status indicators
- Dark mode for accessibility discussion
- Privacy notice explaining that no voice recording is collected

## Run the prototype

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually:

```text
http://localhost:5173/
```

## Video demonstration suggestions

1. Show the project open in VS Code.
